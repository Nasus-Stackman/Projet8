Mon Vieux Grimoire - Back-end

Ce projet consiste en la création d'un serveur backend avec Node.js
L'objectif est non seulement de pouvoir gérer les interactions avec la base de donnée, mais également de permettre aux utilisateurs un connexion sécurisée et authentifiée.
C'est pourquoi les outils utilisés ici sont une APIrestful, la base de donnée noSQL MongoBD. Json Web Token est utilisé au profit de l'auhentification, tandis que l'algorithme de hachage bcrypt est présent pour la sécurisation des mots de passe.
