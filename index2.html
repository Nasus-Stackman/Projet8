Kasa - Application Front-end

Ce projet consiste en la création d'une application Front-end avec react, où différents logements sont consultables en 1 simple click. Les informations proviennent d'un fichier JSON (notaions, lieux, photos, etc...), et sont générés dynamiquement via un langage JSX.
Parmis les contraintes de l'application, on retrouve la création d'un carrousel, le besoin d'une navigation dynamique pour les fiches logements ainsi que la redirection automatique vers une page d'erreur si nécessaire.
Pour se faire, j'utilise ici React-Router, les opportunités des composants React, les States pour la navigation ou les changements d'états, ainsi que le SCSS pour le style.
